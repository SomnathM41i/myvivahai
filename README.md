# 💍 myvivahai

AI-powered matrimonial biodata extraction and matching platform.

**Migration from:** [BioData-AI](https://github.com/SomnathM41i/BioData-AI)

## Stack
- **Backend:** FastAPI + SQLAlchemy (async) + Groq LLaMA AI
- **Frontend:** React 18 + Vite + Tailwind CSS
- **AI Pipeline:** PDF/DOCX/OCR extraction → Groq LLM → structured profile
- **Auth:** Google OAuth 2.0 + JWT
- **Background:** Celery + Redis
